2.3.2. クラス¶

class Man:
    def __init__(self, name):
        self.name = name
        print("Initialized! name = " + name)

    def hello(self):
        print("Hello " + self.name + "!")

    def gooodbye(self):
        print("Good-bye " + self.name + "!")


m = Man("David")
m.hello()
m.goodbye()
